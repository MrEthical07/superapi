# Performance Assets

This folder contains load-testing and benchmarking assets for SuperAPI.

## Structure

- `performance/k6/` — k6 scenario for 10K RPS readiness profile.
- `performance/vegeta/` — Vegeta runner and report generation.
- `performance/results/` — runtime outputs (recommended gitignored locally).

Use `docs/performance-testing.md` as the runbook.
